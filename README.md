# AI功能网站

一个高端科技风格的AI功能网站，提供多种AI应用服务。

## 功能特性

- 🔐 用户注册登录系统
- 👤 个人中心管理
- 💰 积分系统与充值功能
- 🤖 多种AI功能应用
- 🎨 高端科技风格UI设计
- 📱 响应式设计

## 技术栈

### 前端
- React 18
- TypeScript
- Tailwind CSS
- Ant Design
- Axios

### 后端
- Spring Boot 3.x
- Spring Security
- Spring Data JPA
- MySQL
- JWT认证

## 项目结构

```
ai-website/
├── frontend/          # React前端
├── backend/           # Spring Boot后端
└── README.md
```

## 快速开始

### 后端启动
```bash
cd backend
./mvnw spring-boot:run
```

### 前端启动
```bash
cd frontend
yarn install
yarn start
```
